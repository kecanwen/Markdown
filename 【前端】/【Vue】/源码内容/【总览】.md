1. **变化侦测篇**

   学习`Vue`中如何实现数据的响应式系统，从而达到数据驱动视图。

2. **虚拟 DOM 篇**

   学习什么是虚拟 DOM，以及`Vue`中的`DOM-Diff`原理

3. **模板编译篇**

   学习`Vue`内部是怎么把`template`模板编译成虚拟`DOM`,从而渲染出真实`DOM`

4. **实例方法篇**

   学习`Vue`中所有实例方法(即所有以`$`开头的方法)的实现原理

5. **全局 API 篇**

   学习`Vue`中所有全局`API`的实现原理

6. **生命周期篇**

   学习`Vue`中组件的生命周期实现原理

7. **指令篇**

   学习`Vue`中所有指令的实现原理

8. **过滤器篇**

   学习`Vue`中所有过滤器的实现原理

9. **内置组件篇**

   学习`Vue`中内置组件的实现原理
